CSS Flexible Box

Responsive boxes

By adding flex-wrap: wrap;, boxes can flow onto multiple lines.

The flex-basis (e.g. 200px) can either grow (flex-grow) or shrink (flex-shrink).


.row {
	display: flex;
	flex-direction: row;
	flex-wrap: wrap;
}

.row div {
	flex: 1 1 200px;
}
